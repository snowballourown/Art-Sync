② 슬롯 생성 (선생님)

sequenceDiagram actor T as 선생님 (Browser) participant SSC as SpaceSlotController participant TSS as TimeSlotService participant SS as SpaceService participant TSR as TimeSlotRepository note over T: 수업(Space) 선택 후 슬롯 등록 T->>SSC: POST /api/spaces/{spaceId}/slots
{date, startTime, endTime, capacity} SSC->>TSS: createSlot(userId, spaceId, date, start, end, capacity) TSS->>SS: requireOwner(userId, spaceId) SS-->>TSS: 소유자 확인 완료 alt 시작 >= 종료 TSS-->>SSC: BusinessException "시작 시간은 종료 시간보다 빨라야" SSC-->>T: 400 Bad Request end TSS->>TSR: countOverlapping(spaceId, date, start, end) TSR-->>TSS: count alt count > 0 TSS-->>SSC: BusinessException "겹칩니다" SSC-->>T: 400 Bad Request end TSS->>TSR: save(new TimeSlot) [active=false] TSR-->>TSS: saved (id) TSS-->>SSC: slotId SSC-->>T: 201 {id} note over T: 슬롯 공개 (참가자에게 노출) T->>SSC: PATCH /api/spaces/{spaceId}/slots/{id}/activate SSC->>TSS: activate(userId, spaceId, slotId) TSS->>TSR: slot.activate() [active=true] SSC-->>T: 200 OK

③ 예약 신청 · 수락 · 거절

sequenceDiagram actor P as 참가자 (Browser) actor T as 선생님 (Browser) participant RC as ReservationController participant SRC as SpaceReservationController participant RS as ReservationService participant TSS as TimeSlotService participant RR as ReservationRepository participant NS as NotificationService note over P: [예약 신청] P->>RC: POST /api/reservations
{slotId, memo} RC->>RS: request(slotId, memberId, memo) RS->>TSS: getSlot(slotId) TSS-->>RS: TimeSlot alt 비공개 슬롯 RS-->>RC: BusinessException "아직 공개되지 않은 시간대" RC-->>P: 400 end alt 마감 (예약일 전날 23:59 초과) RS-->>RC: BusinessException "예약이 마감되었습니다" RC-->>P: 400 end alt 정원 초과 RS-->>RC: BusinessException "정원이 모두 찼습니다" RC-->>P: 400 end alt 동일 슬롯 중복 신청 RS-->>RC: BusinessException "이미 신청한 예약이 있습니다" RC-->>P: 400 end alt 같은 날 시간 겹침 RS->>RR: countOverlappingForMember() RR-->>RS: count > 0 RS-->>RC: BusinessException "겹치는 예약이 이미 있습니다" RC-->>P: 400 end RS->>RR: save(new Reservation) [REQUESTED] RR-->>RS: reservationId RS->>NS: create(ownerId, RESERVATION_REQUESTED, message) NS-->>T: 알림 저장 RS-->>RC: reservationId RC-->>P: 201 {id} note over T: [예약 수락] T->>SRC: PATCH /api/spaces/{spaceId}/reservations/{id}/confirm SRC->>RS: confirm(userId, reservationId) RS->>TSS: getSlot(slotId) RS->>NS: requireOwner(userId, spaceId) 확인 RS->>RR: countBySlotIdAndStatus(CONFIRMED) alt 정원 초과 RS-->>SRC: BusinessException SRC-->>T: 400 end RS->>RR: reservation.confirm() [CONFIRMED] RS->>TSS: slot.markFull() (정원 찼을 때) RS->>NS: create(memberId, RESERVATION_CONFIRMED, message) NS-->>P: 알림 저장 SRC-->>T: 200 OK note over T: [예약 거절] T->>SRC: PATCH /api/spaces/{spaceId}/reservations/{id}/reject
{reason} SRC->>RS: reject(userId, reservationId, reason) RS->>RR: reservation.reject(reason) [REJECTED] RS->>NS: create(memberId, RESERVATION_REJECTED, message) NS-->>P: 알림 저장 SRC-->>T: 200 OK
